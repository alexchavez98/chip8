# Chip8

CHIP8 emulator written in C (still in progress)

### TO-DO:
- Add sound

### Compile
SDL2 is required

```
gcc -lSDL2 chip-8.c -o ./chip8
```


### Execution
Execute through terminal ./chip and add ROM path as a parameter to execute:

```
./chip8 /path/{ROM}
```
